The first two courses I took revolved around HTML 5 and CSS 3 respectively. However neither are considered a programming language, but a structure/style languages.

What exactly is HTML & CSS ?

Put simple, Hypertext Markup Language & Cascading Style Sheets are two of the core technologies for building Web pages. HTML provides the structure of the page, while CSS provides the visual and positional layout. When asked, I like to explain Web Development as building a house, where HTML plays the function of the raw material used in the building, and CSS plays the function of all of the visual and positional layouts. As what's stone and glass without the right plan to put them together ?

The first thing we learn how to visualize is a basic "Navigation Bar". Something you've dealt with on your daily web surfing or even application use. How it's done is somewhat very simple and could take only seconds, literally... You just add whatever you want using your HTML, slam "display: flex;" on your parent element, add few things that sound complicated ( I'm looking at you "jUsTifY cOnTeNt & AliGn iTeMs") and boom, you got yourself a simple, yet functional, navigation bar.

The Power Of HTML Attributes

As I said before HTML is somewhat of the material used in the building process, so think of attributes as an enhancement for these material. Bullet proof glass, sturdy rock, sandier sand ? I have no idea what they use in construction atp I'm sorry. Anyways, different HTML elements have different attributes, however attributes effects are best shown in forms, including every sign up & application form you've ever dealt with.

With attributes mixed with forms we can indicate the type of input we want, thus forcing you to follow a certain format in the different type of textfields given. Morever these attributes dictate the type of keyboard to pop on your phone whenever engaged with. Ofcourse there's more to it than the "type" attribute, however it's the most engaging with the user.

CSS Animations

Other than styling elements, I also learnt how to transform and animate HTML elements through CSS. With the usage of keyframes you can animate almost anything in any way imaginable. Loops, delays, rotations, even 3D scaling if you ever felt like it, it's all made possible with the help of CSS.

Ofcourse there's more to animations than these simple examples, and by more I mean MORE. From animations related to user interaction that you've definitely seen before, to full on visual animations created to display skill and the limitless possibilities of CSS.
