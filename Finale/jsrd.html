The Ease Of Responsive Design

The last course I took revolved around Responsive Design, which is pretty self explanatory. It focused on the concept of the responsivity of content on different screens, thus adapting web applications to all screen sizes there is. We mainly use CSS to achieve this concept and specifically use Media-Queries to select screen sizes and adjust content as needed.

Bootstrap Framework

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

With the help of bootstrap classes, a navigation bar would only take mere seconds to finish, and is openly customizable in custom CSS code later on

Same goes for buttons, adding multiple bootstrap classes would customize the element even more rendering custom code "not that important"

And yes luckily for all Web Devs in the world, Bootstrap table classes exists, making it soo much easier to lay content and adjust it with just adding classes to Html elements
